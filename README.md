# Stage 01 Project Frontend

## Description

This is stage 01 web app frontend project that uses Typescript, ReactJS, Axios, and Redux. This project implement atomic design & reusable component.

## How to run project

- npm install
- npm run build
- serve -s build

## Features

- Authentication
- Admin page
- List of news
- Create news
- Edit news
- Filter news
- Purchase subscription
